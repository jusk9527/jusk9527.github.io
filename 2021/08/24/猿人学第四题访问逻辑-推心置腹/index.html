理解作者意图

看懂这几行代码我感觉有点懂作者的意思了

  1. 计算返回的key与value的base64组成的md5值
  2. 将css为这个hash值的类先全变为display:none
  3. 从style的left值从而计算排列的值

继续观察<td></td> 内的代码

发现在符合条件的img当中，left属性加一起来都是为0.而且出现最多出现-11.5 这个数字。且其他多为11.5的倍数

假设算法

left为0时位置不变。如果值不是为0 时那么他的位置就是/11.5加上他的下标位置

测试下代码

num_list = [[8, 23.0], [6, -11.5], [1, 11.5], [0, -23.0]]			#
new_num_list = num_list.copy()
for i, tup in enumerate(num_list):
    index = int(tup[1] / 11.5)
    if index != 0:
        new_num_list[i + index] = num_list[i]
num_str = ''.join([str(num[0]) for num in new_num_list])

print(num_str)

def Jkey(key, value):
    j_key = hashlib.md5(base64.b64encode((key + value).encode()).replace(b"=", b"")).hexdigest()
    return j_key


def sumPageNum(pageNum):
    headers = {
        'Connection': 'keep-alive',
        'Accept': 'application/json, text/javascript, */*; q=0.01',
        'User-Agent': 'yuanrenxue.project',
    }
    url = "http://match.yuanrenxue.com/api/match/4?page={}".format(pageNum)
    response = requests.get(url, headers=headers)
    info = response.json().get("info")
    key = response.json().get("key")
    value = response.json().get("value")
    j_key = Jkey(key, value)

    selector = Selector(info)
    td_list = selector.xpath('//td')
    sum_per_page = 0
    for td in td_list:
        img_list = td.xpath('./img')
        num_list = []
        for img in img_list:
            img_number = img.xpath('./@class')[0].extract().replace('img_number ', "")
            src = img.xpath('./@src')[0].extract()
            if j_key != img_number:
                number = int(map_dict.get(src))
                valueLeft = img.xpath('./@style')[0].extract().replace("left:", "").replace("px", "")
                num_list.append([number, float(valueLeft)])
        new_num_list = num_list.copy()
        for i, tup in enumerate(num_list):
            index = int(tup[1] / 11.5)
            if index != 0:
                new_num_list[i + index] = num_list[i]
        num_str = ''.join([str(num[0]) for num in new_num_list])
        num_int = int(num_str)
        sum_per_page += num_int
    return sum_per_page
